# ubuntu-ai-assistant

本项目旨在开发一个专门针对 Ubuntu 24.04.4 LTS 的 Skill，协助新手在通过 Windows 远程桌面 (RDP) 访问的环境下，配置和使用 opencode 与 openclaw 进行 AI 开发。

## 进化铁律 (The Iron Rule of Evolution)
1. **验证优先**: 在任何方案被写入 `SKILL.md` 之前，必须经过用户在真实环境下的验证。
2. **显式确认**: 验证有效后，AI 必须主动询问“是否将此经验进化至 Skill？”，仅在用户明确允许后执行 `edit` 和 `deploy`。
3. **迭代优化**: 若验证无效，AI 必须继续分析并提供替代方案，直到问题解决或用户终止。
4. **拒绝盲目进化**: 严禁在未确认方案有效性的情况下预先修改 Skill 文件。
